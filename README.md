# Taller Grupal Angular

Este taller grupal se realizó en cooperativo con todos los integrantes del grupo, todos coloboraron en su desarrollo.
Se trabajó el archivo de datos .json como una ruta de archivo "private datosUrl = "/datos/data.json"" 
y no como un servidor local para que se pueda apreciar el delpliege del proyecto en vercel y se visualice 
con el link https://taller-grupal-angular.vercel.app/ que también esta fijada en este repositorio.
